# Ticket Booking App

A React application for booking concert tickets with IP blocking to prevent resellers.

## Features

- IP-based access blocking after first login
- Dashboard showing current concerts
- Simulated payment gateway
- User menu for purchases, tickets, history, and personal info

## Setup

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Open http://localhost:5173

## Note

IP blocking is simulated using localStorage. In a real application, this would require backend implementation.
